About The Institute

WarWatch is a B.Tech Web Engineering project designed to monitor, aggregate, and display critical data regarding global conflicts.

Methodology:
This platform integrates Unit 1 (HTML/CSS structures), Unit 2 (JavaScript DOM manipulation and mapping), and Unit 4 (PHP/MySQL Session management) to create a comprehensive data intelligence dashboard inspired by leading global think-tanks.
